# PROJECT_PLAN.md — CareerFit AI 프로젝트 기획 초안

## 1. 문제 
Q1. 취업 공고나 공모전 목록을 찾아보면서 있었던 문제점은?
- 여러 공고가 흩어져 있다

Q2. CareerFit AI가 이 문제를 어떻게 해결하는가?
- 사용자가 원하는 회사,직무,기술,경험을 키워드로 입력하면
- 채용공고 데이터베이스에서 관련 공고를 검색하여 관련성이 높은 순서대로 표시한다.
- 각 공고의 핵심 내용을 빠르게 파악할 수 있도록 요약해준다.
- 공고와 사용자 경험을 직관적으로 비교할 수 있도록 인터페이스를 제공한다.



Q3. 이 서비스를 쓸 사용자가 누구인가?
- 취업 준비 중인 3~4학년


## 2. 사용자 정의
- 대상: 취업을 준비하는 3~4학년
- 사용 시나리오:
1. 사용자가 전공, 보유 스킬, 관심 직무를 입력한다.
2. CareerFit AI가 데이터를 검색해 역량 매칭 결과를 반환한다.
3. 어떤 데이터를 기반으로 답변했는지 출처를 함께 표시한다.


## 3. 데이터 계획
- 취업 공고 컬럼: 회사명, 직무명, 요구 스킬, 우대 사항, 지원 자격, 업무 내용, 마감일
- 공모전 컬럼: 공모전명, 분야, 대상, 주최, 접수 기간, 발표일, 내용, 링크
- 데이터 출처: 강사 제공 목업 CSV (3일차)
